To install dependencies:
```sh
bun install
```

To run:
```sh
bun run dev
```

<h2>API DOCS</h2>

### Homepage Endpoint
```
/main/api/service
```

### Search Anime
```
/main/api/service/search?q=
```

### Get Anime Episode 
```
/main/api/service/:animetitle
```

### Get Anime Genre
```
/main/api/service/otakudesu/genre/:genreAnime/:page
```

### Get Anime Stream Source
```
/main/api/service/otakudesu/play/:animeTitleWithEpisode
```
